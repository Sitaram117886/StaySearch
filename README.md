# StaySearch

> A modern hotel discovery and booking web application built with HTML, CSS, and JavaScript.

StaySearch is a front-end hotel booking application that allows users to search hotels by city or budget, explore hotel details, manage a wishlist, book stays with live price calculation, and view booking history through a clean and modern user interface.

The project focuses on delivering a smooth user experience with thoughtful UI design, reusable components, and interactive animations while demonstrating strong front-end development fundamentals.

---

## Features

- Smart hotel search by city or hotel name
- Browse hotels by popular cities
- Budget-based hotel discovery
- Detailed hotel information page
- Wishlist management using Local Storage
- Complete hotel booking flow
- Guest and room selection
- Live booking price calculation
- Booking history with previously booked badges
- Sort hotels by price, rating, and name
- Custom loading animations
- Scroll-to-top button
- Interactive project roadmap panel
- Responsive design for different screen sizes

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage
